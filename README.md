# Telegram-бот FAQ для СДН ПНУ

Університетський проєкт. Telegram-бот з відповідями на найчастіші запитання щодо роботи у Системі дистанційного навчання (d-learn.pnu.edu.ua).

Бот дозволяє:

- Переглянути списки питань для студентів і викладачів та одразу отримати відповідь.
- Переглянути теми з методичної інструкції для викладачів (із зображеннями зі скріншотами з PDF).
- Поставити власне запитання текстом — бот шукає найближчу відповідь у базі. Якщо відповіді немає, повертає посилання на офіційний FAQ та YouTube-канал.

## Стек

- Node.js 20+
- [grammY](https://grammy.dev/) — бібліотека для Telegram-ботів
- [Fuse.js](https://www.fusejs.io/) — пошук по запиту користувача
- `dotenv` — конфігурація через `.env`

## Структура проєкту

```
tg-bot-university/
├── src/
│   ├── bot.js                 точка входу
│   ├── handlers/
│   │   ├── menu.js            головне меню та підменю
│   │   ├── faq.js             відправка відповіді з фото
│   │   └── search.js          обробник вільного тексту
│   └── search/index.js        конфіг Fuse.js та пошук
├── data/faq.json              усі питання, відповіді, посилання та фото
├── assets/images/             скріншоти з PDF
├── scripts/extract-images.js  допоміжний скрипт для витягу зображень з PDF
├── .env.example
├── package.json
└── README.md
```

## Як отримати токен бота

1. У Telegram знайдіть `@BotFather`.
2. Надішліть `/newbot`.
3. Введіть назву (наприклад, `PNU FAQ Bot`).
4. Введіть username (має закінчуватися на `bot`, наприклад `pnu_faq_bot`).
5. BotFather надішле повідомлення з токеном виду `123456789:AA...`. Скопіюйте його.

## Запуск

```bash
npm install
cp .env.example .env
# відкрийте .env та вставте свій BOT_TOKEN
npm start
```

Для розробки з автоматичним перезапуском:

```bash
npm run dev
```

## Як додати зображення з PDF

PDF з методичкою кладеться у `assets/source.pdf`. Потім:

```bash
npm run extract-images
```

Скрипт згенерує зображення сторінок у `assets/images/page-XX.png`. Прив'язка сторінок до питань вже зроблена у `data/faq.json` у полі `images`.

## Як додати нове питання

Відкрити `data/faq.json` і додати новий запис у потрібну категорію:

```json
{
  "id": "students.new_topic",
  "category": "students",
  "question": "Коротка назва",
  "answer": "Повна відповідь у Markdown-стилі.",
  "keywords": ["ключові", "слова", "для", "пошуку"],
  "images": ["assets/images/page-12.png"],
  "links": []
}
```

Бот підхопить це після перезапуску.
